JSON - Evaluates to JavaScript Objects

Định dạng JSON gần giống với các đối tượng JavaScript. Trong JSON, các khóa phải là chuỗi, được viết bằng dấu ngoặc kép

JSON Values

Trong JSON, các giá trị phải là một trong các loại dữ liệu sau: string, number, object, array, boolean, null Trong JS các giá trị có thể là tất cả các giá trị trên, cộng với bất kỳ biểu thức JavaScript hợp lệ nào khác: function, date, undefined

JavaScript Objects

Vì cú pháp JSON bắt nguồn từ ký hiệu đối tượng JavaScript nên cần rất ít phần mềm bổ sung để hoạt động với JSON trong JavaScript. Với JavaScript, bạn có thể tạo một đối tượng và gán dữ liệu cho nó
